## Installation and Running App 

To install dependencies, run:

```
pip3 install -r requirements.txt
```

To run the server:

```
python3 app.py
```


## Technologies used
- [python](https://www.python.org)
- [Flask](https://flask.palletsprojects.com/en/2.2.x/)
- [SqlAlchemy](https://www.sqlalchemy.org)
